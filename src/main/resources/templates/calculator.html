Free-form Operations:

Known issues with the free-form functionality:

  • The calculation only supports single digit numbers.
  • The calculation doesn't support parentheses or exponents.
  • The order of operations is not exact. Multiplication always comes before Division and Addition always comes before Subtraction in the code.
  • The calculation doesn't work when doing the same operation multiple times (i.e. 4+4+4+4).
